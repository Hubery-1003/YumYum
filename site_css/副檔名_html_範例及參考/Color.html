使用範例：

p {
    color: var(--yum-primary-2);
}
將 p 標籤的文字顏色設定成 --yum-primary-2 的顏色

button {
    background-color: var(--yum-secondary-1);
}
將 button 標籤的背景顏色設定成 --yum-secondary-1 的顏色
